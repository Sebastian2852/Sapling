# Sapling Framework

The sapling framework is a collection of packages to help speed up development on roblox. Most of the packages expect a fairly
moderate understanding of the Luau programming language.

Sapling Framework is expected to be installed by [Pesde](https://pesde.dev) using [Git Dependencies](https://docs.pesde.dev/guides/dependencies/#git-dependencies).

## What is it?

The sapling framework is a collection of pacakges that aim to help you speed up development on roblox. Sapling Framework is inspired by
nevermoer engine and aims to provide such ease of use but with more advanced features.

## Why Sapling Framework?

- ***Bloat Free:*** You only install what you need to use, no extra fluff needs to be downloaded to use anything. Packages aim to use as little dependencies as possible.
- ***Simple & Easy to learn:*** The Sapling Framework aims to provide simple APIs understandable to anyone which also providing a power and flexible platform for developing Roblox games.
- ***Its YOURS:*** Everything is as customisable as can be to allow you to tailor the framework to your game how you want without modifing the source code of the package.
